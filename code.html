The Code

                        
                            // get values from the page
                            //controller function
                            function getValues() {
                              //get values from the page
                              let startValue = document.getElementById("startValue").value;
                              let endValue = document.getElementById("endValue").value;
                            
                              startValue = parseInt(startValue);
                              endValue = parseInt(endValue);
                            
                              if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                                //we call generateNumbers
                                let numbers = generateNums(startValue, endValue);
                                //we call displayNumbers
                                displayNums(numbers);
                              } else {
                                alert("You must enter a number");
                              }
                            }
                            
                            // generate numbers from startvalue to the endvalue
                            //logic function
                            function generateNums(startValue, endValue) {
                              let numbers = [];
                            
                              for (let index = startValue; index <= endValue; index++) {
                                numbers.push(index);
                              }
                            
                              return numbers;
                            }
                            
                            // display the numbers, mark even numbers bold
                            // display or view function
                            function displayNums(numbers) {
                              let templateRows = "";
                              for (let index = 0; index < numbers.length; index++) {
                                let className = "even";
                                let number = numbers[index];
                                if (number % 2 == 0) {
                                  className = "even";
                                } else {
                                  className = "odd";
                                }
                                //  prismjs does not support syntax highlighting for template literals
                                templateRows += `${number}`;
                              }
                            
                              document.getElementById("results").innerHTML = templateRows;
                            }

The code is structured in three functions

The Get Values Function

The getValues function is used to get the start and end values by parsing the document for the id of startValue and endValue respectively. Once we have access to the start and end values, we use the parseInt function to convert the numbers from strings to integers. An if statement is added to check the validity of the input that is received. If a number is not entered for both values, the user will be alerted to enter a number.

The Generate Nums Fuction

We begin the generateNums function by assigned the value of numbers to an empty array. This function takes in two parameters, startValue and endValue. We use a for loop to loop over the input and push each number including the endValue into the array array. Once the loop is completed. The function returns the numbers. This function is called as we check the validity of the numbers using the if statement located in the getValues functions

The Display Nums Function

The function displayNums takes one argument, numbers. We begin the function by assigning the variable templateRows to an empty string. In this function a for loop is used to iterate over the value to determine if the number is even or odd. Within the for loop, the className variable is set to the value of even. Within our for loop we have an if statement that validates whether a number is even or odd using the formula number % 2 == 0. The modulus or % returns the remainder of a divison operation. Any number with a modulus of 2 that is equal to 0 is an even number. If the number does not pass the even number validation and else statement is used to identify the number as odd. We use string concatenation to dynamically add the class name to the table data and display the number in the table. Finally use the getElementById function to find the results id and assign the innerHTML to the templateRows variable. Once all validation has completed, the displayNums fuction is executed after the numbers have been assigned using the generateNums function.
